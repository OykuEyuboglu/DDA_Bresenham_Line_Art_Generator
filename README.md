## 🌟 Project Description

This project demonstrates two fundamental **computer graphics line drawing algorithms**:

- **DDA (Digital Differential Analyzer)**
- **Bresenham Line Algorithm**

The implementation is written in **Python** using the **Pygame** graphics library and developed in a **Jupyter Notebook environment**.

The program generates random colored lines and sparkle effects on a pixel-based screen.  
This creates a simple animated visual output while demonstrating how both algorithms generate lines.

---

## 🎯 Project Purpose

The main goal of this project is to understand how **lines are drawn on pixel-based displays** and to observe the differences between the DDA and Bresenham algorithms.

By combining algorithm implementation with simple visual effects, the project provides both:

- a **technical understanding of line drawing algorithms**
- a **visual demonstration through animation**

---

## ⚙️ Program Behavior

When the program runs:

- A graphical window is created using **Pygame**
- The screen background is cleared
- Random sparkle effects appear on the screen
- Random colored lines are generated
- Lines are drawn using **both algorithms alternately**

Because the coordinates and colors are generated randomly, every execution produces **different visual patterns**.

---

## 🧠 Algorithms Used

### 📏 DDA Algorithm

The **Digital Differential Analyzer (DDA)** algorithm calculates intermediate points between two coordinates using floating-point arithmetic.

Characteristics:

- Simple and easy to implement  
- Uses incremental calculations  
- Demonstrates basic pixel-based line drawing

---

### 📐 Bresenham Algorithm

The **Bresenham Line Algorithm** determines the next pixel using integer calculations and a decision parameter.

Characteristics:

- More efficient than DDA  
- Uses integer arithmetic  
- Widely used in computer graphics systems
